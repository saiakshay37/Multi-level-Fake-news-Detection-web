Pipeline

How It Works

Six stages .

📥
Input Ingestion
Accepts headlines, paragraphs, or article URLs. For URLs, newspaper3k extracts content with BeautifulSoup as fallback (Cell 18).
🔧
NLP Preprocessing
lowercase → strip URLs/numbers/punctuation → Porter stemming → remove stopwords. Negations (not, never, nor) are preserved.
🗂️
Fact DB Override
35 hard-coded facts checked by keyword overlap ≥ 60%. Instant FAKE/REAL verdict — model is skipped for these known claims.
🧠
CNN + Bi-LSTM
Two Conv1D layers (128+64 filters) extract n-gram features. Two Bidirectional LSTMs capture context. Sigmoid outputs P(REAL).
📊
4-Level Mapping
P(fake)≥0.75→FAKE · ≥0.55→MISLEADING · P(real)≥0.75→REAL · else→PARTIALLY REAL. Thresholds FAKE_HIGH, FAKE_LOW, REAL_HIGH from Config.
🔍
Explainable AI
LIME highlights which words drove the prediction. Occlusion fallback when LIME fails. ↑ red = pushes toward FAKE · ↓ green = toward REAL.
Classification System

Beyond Binary

Four levels reflecting how misinformation actually exists — from outright fabrication to borderline reporting.

🔴
FAKE
Deliberately fabricated. Sensational language, conspiracy signals, or matches a Fact DB entry. P(fake) ≥ 0.75 from the CNN + Bi-LSTM model (cfg.FAKE_HIGH).
🟠
MISLEADING
May have a factual kernel but uses deceptive framing or missing context to create a false impression. 0.55 ≤ P(fake) < 0.75 (cfg.FAKE_LOW).
🟡
PARTIALLY REAL
Genuinely borderline — model is uncertain. Typically early breaking news, unconfirmed claims, or content mixing real and speculative elements.
🟢
REAL
Verified, well-sourced reporting with named officials, institutional attribution, or data citations. P(real) ≥ 0.75 from model (cfg.REAL_HIGH).

Training Data

WELFake handles US political/media fake news. The custom dataset covers Indian, health, science and government misinformation — preventing the model from labelling "coffee cures cancer" as REAL.

Model Architecture

CNN + Bi-LSTM

Binary sigmoid output — two Conv1D layers (128+64 filters), vocab=20K, 8 epochs, early stopping. P(REAL) from sigmoid is mapped to 4 levels using Cell 16 thresholds.
